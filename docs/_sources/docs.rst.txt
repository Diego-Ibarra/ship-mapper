Build
-----
.. warning::

    The following information is **only** for developers and maintainers of docs. 

Everytime you update the `.rts` documentation source files, you will need to run this in the console...

::

    sphinx-build -E -a -b html C:\Users\IbarraD\ship_mapper\docs\sphinx C:\Users\IbarraD\ship_mapper\docs
    
    
Note: Replace the paths to reflect your computer
