VMS_DFO_Oracle Converter¶

VMS_DFO_Oracle.convert(file_in, file_out, data_info_file)[source]¶

This is a csv to nc converter for VMS data

Parameters:
  • file_in (str) – Input file (.csv)
  • file_out (str) – Output file (.nc)
  • data_info_file (str) – YAML file describing other conversion parameters (.yml)
Returns:

A netCDF file (.nc) in a format standard for “ship_mapper” containing the original data, but also containing metadata included in the data_info.yml file
